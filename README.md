⌨️ StepTyper — Symbol-by-Symbol Code Injector for Visual Studio
🎯 Нажимайте любую клавишу → в Visual Studio появляется ровно один следующий символ из вашего файла.
🔒 Ваша клавиша не дублируется в редакторе (полное подавление ввода).

⚠️ Устранение неполадок
Проблема
Решение
❌ "Нужны права администратора"
Запустите терминал/программу правым кликом → "От имени администратора"
❌ Вводятся "кривые" символы (б вместо {)
Переключите раскладку клавиатуры на EN-US перед запуском
❌ Скрипт не видит файл PythonApplication1
Убедитесь, что файл лежит в той же папке, что и скрипт. Попробуйте добавить расширение .txt
❌ Visual Studio "тормозит" при вводе
Увеличьте задержку в коде: time.sleep(0.015) → 0.03 или временно отключите IntelliSense
❌ Нажатая клавиша всё равно появляется в VS
Убедитесь, что запустили от администратора. Без этого suppress=True не работает на Windows
❌ Антивирус блокирует .exe
Добавьте папку в исключения. Ложное срабатывание из-за перехвата клавиш — обычное дело
🛡️ Безопасность и приватность
🔐 Скрипт не отправляет ваши данные в интернет.
📁 Весь ввод обрабатывается локально.
⚠️ Библиотека keyboard требует прав администратора для низкоуровневого перехвата — это стандартное поведение для кейлоггеров и макросов. Используйте только доверенные сборки.
